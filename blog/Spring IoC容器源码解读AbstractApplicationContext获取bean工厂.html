# Spring IoC容器 源码解析——ApplicationContext获取bean工厂

AbstractApplicationContext使用getBeanFactory获取bean工厂，用于注册bean，实例化bean。该方法使用abstract令子类覆盖实现。

public abstract ConfigurableListableBeanFactory getBeanFactory() throws IllegalStateException;

Spring提供GenericApplicationContext实现AbstractApplicationContext，该类维护一个DefaultListableBeanFactoru实例，用于提供bean工厂。

该类会在构造时，直接构建bean工厂。

public GenericApplicationContext() {
    this.beanFactory = new DefaultListableBeanFactory();
}

GnericApplicationContext通过直接返回bean工厂实例，实现getBeanFactory()方法。

@Override
public final ConfigurableListableBeanFactory getBeanFactory() {
    return this.beanFactory;
}
