# java HashMap treeifyBin操作

HashMap执行put操作时，如果表内特定索引内的节点数量大于等于8时，执行树化操作treeifyBin。

# treeifbyBin

首先判断表是否为空，或者表的尺寸是否小于64，如果过小，则执行resize。

if (tab == null || (n = tab.length) < MIN_TREEIFY_CAPACITY)
    resize();

获取给定hash位置的结点，遍历链表。首先将结点转为树节点，如果尾部为null，设置新节点为首部。否则，将新节点放在尾部之后，尾部指针指向新节点。

do {
    TreeNode<K,V> p = replacementTreeNode(e, null);
    if (tl == null)
        hd = p;
    else {
        p.prev = tl;
        tl.next = p;
    }
    tl = p;
} while ((e = e.next) != null);

将新的TreeNode放入表节点中，执行treeify操作。

if ((tab[index] = hd) != null)
    hd.treeify(tab);
