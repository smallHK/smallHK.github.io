# Spring 注解 TypeMappedAnnotation创建特定注解

TypeMappedAnnotation使用工厂方法实例化MergedAnnotation。

调用时，直接实例化。

return new TypeMappedAnnotation<>(mapping, null, source, rootAttribute,
        valueExtractor, aggregateIndex);

如果实例化过程出现异常，则执行配置异常再抛出。

AnnotationUtils.rethrowAnnotationConfigurationException(ex);

# TypeMappedAnnotation实例化

TypeMappedAnnotation实例化涉及多个字段的赋值。

设置类加载器

this.classLoader = classLoader;

设置源

this.source = source;

设置根属性

this.rootAttributes = rootAnnotation;

设置值提取器

this.valueExtractor = valueExtractor;

设置注解类型映射

this.mapping = mapping;

设置聚合索引

this.aggregateIndex = aggregateIndex;

设置使用合并值

this.useMergedValues = useMergedValues;

设置属性过滤器

this.attributeFilter = attributeFilter;

设置已解析根镜像

this.resolvedRootMirrors = resolvedRootMirrors;

设置已解析镜像

this.resolvedMirrors = resolvedMirrors;
