# Spring IoC AnnotatedGenericBeanDefinition实例化

AnnotatedGenericBeanDefinition为BeanDefinition的注解实现。

AnnotatedGenericBeanDefinition的实例化一共分两步，第一步设置bean的Class，第二步解析注解元信息。

setBeanClass(beanClass);
this.metadata = AnnotationMetadata.introspect(beanClass);

AnnotationMetadata为接口，内置了静态工厂方法用于创建类的注解元信息。

static AnnotationMetadata introspect(Class<?> type) {
    return StandardAnnotationMetadata.from(type);
}

StandardAnnotationMetadata为AnnotationMetadata的实现，使用反射方法检查类。

public class StandardAnnotationMetadata extends StandardClassMetadata implements AnnotationMetadata

该类为解析注解元信息工厂的方法的实际实现。

static AnnotationMetadata from(Class<?> introspectedClass) {
    return new StandardAnnotationMetadata(introspectedClass, true);
}

# StandardAnnotationMetadata实例化

该实现维护一个MergedAnnotations实例，用于存储方法注解信息。实例化过程即通过解析类，获取对应的元信息。

super(introspectedClass);
this.mergedAnnotations = MergedAnnotations.from(introspectedClass,
        SearchStrategy.INHERITED_ANNOTATIONS, RepeatableContainers.none(),
        AnnotationFilter.NONE);
this.nestedAnnotationsAsMap = nestedAnnotationsAsMap;

StanardAnnotationMetadata实例化之前，会调用超类StandardClassMetadata的构造器，该类即类的元信息实现。

@Deprecated
public StandardClassMetadata(Class<?> introspectedClass) {
    Assert.notNull(introspectedClass, "Class must not be null");
    this.introspectedClass = introspectedClass;
}

实例化mergedAnnotations通过MergerdAnnotations的from方法实现。实例化时，扫描所有注解以及继承的注解。

this.mergedAnnotations = MergedAnnotations.from(introspectedClass,
        SearchStrategy.INHERITED_ANNOTATIONS, RepeatableContainers.none(),
        AnnotationFilter.NONE);

MergedAnnotations使用TypeMappedAnnotations的from方法实现注解解析。

return TypeMappedAnnotations.from(element, searchStrategy, repeatableContainers, annotationFilter);

TyepMappedAnnotations为MergedAnnotations的实现。

final class TypeMappedAnnotations implements MergedAnnotations

TypeMappedAnnotations通过实例化自身，实现from方法。

if (AnnotationsScanner.isKnownEmpty(element, searchStrategy)) {
    return NONE;
}
return new TypeMappedAnnotations(element, searchStrategy, repeatableContainers, annotationFilter);

private TypeMappedAnnotations(AnnotatedElement element, SearchStrategy searchStrategy,
        RepeatableContainers repeatableContainers, AnnotationFilter annotationFilter) {

    this.source = element;
    this.element = element;
    this.searchStrategy = searchStrategy;
    this.annotations = null;
    this.repeatableContainers = repeatableContainers;
    this.annotationFilter = annotationFilter;
}

TypeMappedAnnotations会在实例化过程中，进行逐个实例字段赋值。
