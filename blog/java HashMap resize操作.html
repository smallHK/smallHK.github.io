# java HashMap resize操作

HashMap使用resize操作扩容表，或初始化表。

# resize

获取旧表尺寸，以及旧阈值，初始化新表尺寸，新表阈值。

Node<K,V>[] oldTab = table;
int oldCap = (oldTab == null) ? 0 : oldTab.length;
int oldThr = threshold;
int newCap, newThr = 0;

如果旧表尺寸大于0，如果旧表尺寸大于等于最大阈值2^30，将于是设置为Integer的最大值，返回旧表。如果旧表尺寸小于阈值，则新尺寸*2，如果新尺寸小于最大尺寸，并且旧尺寸大于模式尺寸尺寸16，新的阈值为旧阈值翻倍。

if (oldCap > 0) {
if (oldCap >= MAXIMUM_CAPACITY) {
    threshold = Integer.MAX_VALUE;
    return oldTab;
}
else if ((newCap = oldCap << 1) < MAXIMUM_CAPACITY &&
            oldCap >= DEFAULT_INITIAL_CAPACITY)
    newThr = oldThr << 1; // double threshold
}

如果旧的尺寸为0，旧的阈值大于0，将新的尺寸设置为旧的阈值。

else if (oldThr > 0) // initial capacity was placed in threshold
    newCap = oldThr;

如果旧的阈值与旧的尺寸都为0，设置新尺寸为初始尺寸16，新的阈值为默认负载因子0.75 * 默认初始尺寸16。

else {               // zero initial threshold signifies using defaults
    newCap = DEFAULT_INITIAL_CAPACITY;
    newThr = (int)(DEFAULT_LOAD_FACTOR * DEFAULT_INITIAL_CAPACITY);
}

如果新的阈值为0，求新尺寸*负载因子的值，如果新尺寸小于最大尺寸，并且求解值小于最大尺寸，将求解值设为新的阈值，否则使用Integer最大值。

if (newThr == 0) {
    float ft = (float)newCap * loadFactor;
    newThr = (newCap < MAXIMUM_CAPACITY && ft < (float)MAXIMUM_CAPACITY ?
                (int)ft : Integer.MAX_VALUE);
}

使用新阈值作为阈值。

threshold = newThr;

使用新尺寸构建新表，将新表设置为表。

Node<K,V>[] newTab = (Node<K,V>[])new Node[newCap];
table = newTab;

如果旧表非空，对旧表执行迭代。

if (oldTab != null) {
    for (int j = 0; j < oldCap; ++j) {
        //...
    }
}

判断当前位置节点非null。判断该节点是否为孤立节点，如果是孤立节点，直接放入新表。

if (e.next == null)
    newTab[e.hash & (newCap - 1)] = e;

判断该节点是否为TreeNode，如果是树节点，执行split操作。

else if (e instanceof TreeNode)
    ((TreeNode<K,V>)e).split(this, newTab, j, oldCap);

如果节点为链表，判断如果节点的hash与旧尺寸与操作为0，放入lo链表，否则放入hi链表。

do {
        next = e.next;
        if ((e.hash & oldCap) == 0) {
            if (loTail == null)
                loHead = e;
            else
                loTail.next = e;
            loTail = e;
        }
        else {
            if (hiTail == null)
                hiHead = e;
            else
                hiTail.next = e;
            hiTail = e;
        }
    } while ((e = next) != null);

如果lo非空，将lo链表放入新表旧位置，如果hi非空，将hi链表放入旧尺寸加j位置。

if (loTail != null) {
    loTail.next = null;
    newTab[j] = loHead;
}
if (hiTail != null) {
    hiTail.next = null;
    newTab[j + oldCap] = hiHead;
}

返回修正后的新表。

return newTab;
