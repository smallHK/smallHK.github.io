# Spring IoC容器 源码解析——AbstractApplicationContext刷新

容器初始化时，会执行刷新操作。refresh为ConfigurableApplicationContext中生命的方法。

刷新操作首先会执行预刷新，准备刷新的上下文。

prepareRefresh();

可配置的获取bean工厂。

ConfigurableListableBeanFactory beanFactory = obtainFreshBeanFactory();

准备bean工厂。

prepareBeanFactory(beanFactory);

后处理bean工厂。

postProcessBeanFactory(beanFactory);

调用bean工厂后处理器

invokeBeanFactoryPostProcessors(beanFactory);

注册bean post处理器

registerBeanPostProcessors(beanFactory);

初始化消息源

initMessageSource();

初始化应用事件广播

initApplicationEventMulticaster();

初始化特定beans

onRefresh();

验证监听器，并注册

registerListeners();

结束bean工厂初始化

finishBeanFactoryInitialization(beanFactory);

结束刷新

finishRefresh();

如果期间发生了beans异常，销毁bean，并且取消刷新。

// Destroy already created singletons to avoid dangling resources.
destroyBeans();

// Reset 'active' flag.
cancelRefresh(ex);

最终重置缓存

resetCommonCaches();
