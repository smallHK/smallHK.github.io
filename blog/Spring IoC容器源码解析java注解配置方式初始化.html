# Spring IoC容器 源码解析——注解方式初始化

Spring IoC容器使用注解方法初始化时，可使用初始化AnnotationConfigApplicationContext实例完成。

ApplicationContext ctx = new AnnotationConfigApplicationContext(AppConfig.class);

AnnotationConfigApplicationContext使用组件类初始化时，执行三步操作。

  1. 调用默认构造器
  2. 注册组件类
  3. 刷新容器

# AnnotationConfigApplicationContext默认构造器

默认构造器执行两步操作实例化注解bean定义读取器，实例化类路径bean定义扫描器。

public AnnotationConfigApplicationContext() {
    this.reader = new AnnotatedBeanDefinitionReader(this);
    this.scanner = new ClassPathBeanDefinitionScanner(this);
}

# 注册组件类

注册组件类，将会判断传入的组件类是否为null，或者是否为空。如果为空，抛出非法参数异常。

调用读取器注册方法。

public void register(Class<?>... componentClasses) {
    Assert.notEmpty(componentClasses, "At least one component class must be specified");
    this.reader.register(componentClasses);
}
