# java CompletableFuture执行thenAccept

CompletableFuture使用uniAcceptStage实现。

public CompletableFuture<Void> thenAccept(Consumer<? super T> action) {
    return uniAcceptStage(null, action);
}

# uniAcceptStage

uniAcceptStage首先判断消费者是否为null，如果为null，抛出异常。

if (f == null) throw new NullPointerException();

判断Future维护的result是否为null，如果存在值执行立即返回。

if ((r = result) != null)
    return uniAcceptNow(r, e, f)

实例化一个未完成Future。执行push操作，将新建UniAccept对象推入。随后将其返回。

CompletableFuture<Void> d = newIncompleteFuture();
unipush(new UniAccept<T>(e, d, this, f));
return d;

# uniAcceptNow

首先新建一个未完成CompletableFuture。判断结果是否为AltResult。如果是AltResult，且其ex不为null，将异常封装，然后返回未完成Future。

if (r instanceof AltResult) {
    if ((x = ((AltResult)r).ex) != null) {
        d.result = encodeThrowable(x, r);
        return d;
    }
    r = null;
}

如果结果并非封装的异常，将r设置为null。

r = null;

如果存在指定的执行器，执行器执行UniAccept。否则，直接执行消费者。result设置为封装的NIL。如果期间抛出了异常，封装异常为结果，然后返回。

try {
    if (e != null) {
        e.execute(new UniAccept<T>(null, d, this, f));
    } else {
        @SuppressWarnings("unchecked") T t = (T) r;
        f.accept(t);
        d.result = NIL;
    }
} catch (Throwable ex) {
    d.result = encodeThrowable(ex);
}
