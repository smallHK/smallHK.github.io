# Spring IoC AnnotationScopeMetadataResolver解析scope元数据

AnnotationScopeMetadataResolver用于解析注解bean类的Scope注解。该类实现ScopeMetadataResolver，用于解析生成scope元数据。

新建ScopeMetadata实例。

ScopeMetadata metadata = new ScopeMetadata();

判断输入的BeanDefinition是否为AnnotatedBeanDefinition。通过AnnotationConfigUtils读取Scope类型的注解信息。通过注解信息设置元信息Scope名称，判断是否存在代理模式，如果存在，则设置代理模式。

if (definition instanceof AnnotatedBeanDefinition) {
    AnnotatedBeanDefinition annDef = (AnnotatedBeanDefinition) definition;
    AnnotationAttributes attributes = AnnotationConfigUtils.attributesFor(
            annDef.getMetadata(), this.scopeAnnotationType);
    if (attributes != null) {
        metadata.setScopeName(attributes.getString("value"));
        ScopedProxyMode proxyMode = attributes.getEnum("proxyMode");
        if (proxyMode == ScopedProxyMode.DEFAULT) {
            proxyMode = this.defaultProxyMode;
        }
        metadata.setScopedProxyMode(proxyMode);
    }
}

# AnnotationConfigUtils读取Scope类型的注解信息

AnnotationConfigUtils通过attributesFor获取注解属性。

@Nullable
static AnnotationAttributes attributesFor(AnnotatedTypeMetadata metadata, Class<?> annotationClass) {
    return attributesFor(metadata, annotationClass.getName());
}

attrobutesFor通过AnnotatedTypeMetadata的get方法获取Annotations属性。

@Nullable
static AnnotationAttributes attributesFor(AnnotatedTypeMetadata metadata, String annotationClassName) {
    return AnnotationAttributes.fromMap(metadata.getAnnotationAttributes(annotationClassName, false));
}

getAnnotationAttrobutes由AnnotatedTypeMetadata提供默认实现。该实现通过提供首个直接声明选择策略获取MergedAnnotation。判断获取的注解是否位于源码，如果不位于源码，则返回null，如果位于源码，则转为map返回。

MergedAnnotation<Annotation> annotation = getAnnotations().get(annotationName,
        null, MergedAnnotationSelectors.firstDirectlyDeclared());
if (!annotation.isPresent()) {
    return null;
}
return annotation.asAnnotationAttributes(Adapt.values(classValuesAsString, true));
