# netty NioServerSocketChannel实例化

NioServerSocketChannel允许默认实例化，默认实例化时，使用平台提供的Selector开启ServerSocketChannel。

public NioServerSocketChannel() {
    this(newSocket(DEFAULT_SELECTOR_PROVIDER));
}

执行超类AbstractNioMessageChannel实例化方法。兴趣操作为ACCEPT，使用默认选择器提供者开启的serverSocketChannel。

super(null, channel, SelectionKey.OP_ACCEPT);

超类AbstractNioMessageChannel实例化超类AbstractNioChannel。

super(parent, ch, readInterestOp);

AbstractNioChannel首先实例化超类AbstractChannel。设置Channel的双亲，即null，设置默认通道id，设置unsafe，新建通道pipeline。实例化的管道使用DefaultChannelPipeline。

this.parent = parent;
id = newId();
unsafe = newUnsafe();
pipeline = newChannelPipeline();

AbstractNioChannel将serverSocket通道设置为当前ch字段，设置感兴趣的操作集。将通道设置为非阻塞。如果设置失败，抛出异常。

this.ch = ch;
this.readInterestOp = readInterestOp;
try {
    ch.configureBlocking(false);
} catch (IOException e) {
    try {
        ch.close();
    } catch (IOException e2) {
        if (logger.isWarnEnabled()) {
            logger.warn(
                    "Failed to close a partially initialized socket.", e2);
        }
    }

    throw new ChannelException("Failed to enter non-blocking mode.", e);
}

实力内部类NioServerSocketChannelConfig实例化配置config字段。

config = new NioServerSocketChannelConfig(this, javaChannel().socket());
